Zikpro AI Invoice OCR is an intelligent ERPNext extension that automates invoice processing using AI.

It extracts structured data from invoices (PDFs and images), validates financial data, matches suppliers, and automatically creates Purchase Invoices inside ERPNext.

---

## How It Works

1. Upload invoice (PDF or image)
2. AI extracts invoice data
3. System validates totals and taxes
4. Supplier is matched automatically
5. User reviews extracted data
6. Purchase Invoice is created

---

## Key Features

• AI-powered invoice data extraction  
• Support for PDF and image invoices  
• Automatic supplier detection  
• Line items and tax extraction  
• Financial validation and error detection  
• Automatic Purchase Invoice creation in ERPNext  
• Mobile camera invoice upload  
• Duplicate invoice protection
